🚀 Flutter Cheatsheet

🧹 Очистка проекта

Базовая очистка

flutter clean

Удаление lock-файлов и кэшей

rm -f pubspec.lock rm -rf .dart_tool rm -f .flutter-plugins rm -f .flutter-plugins-dependencies rm -rf .idea

Очистка Android-сборки

rm -rf android/.gradle rm -rf android/build rm -rf android/app/build rm -f android/local.properties

📦 Глобальный Gradle кэш

⚠️ После очистки первая сборка займёт 10-20 минут!

Убить Java/Gradle процессы

taskkill //F //IM java.exe 2>/dev/null taskkill //F //IM gradle.exe 2>/dev/null

Удалить глобальные кэши

rm -rf ~/.gradle/caches rm -rf ~/.gradle/daemon rm -rf ~/.gradle/wrapper/dists

⚙️ Flutter / Dart кэш

dart pub cache repair flutter precache --force flutter doctor

🔧 Build Runner

Разовая генерация

dart run build_runner build --delete-conflicting-outputs

Watch mode (автогенерация при сохранении)

dart run build_runner watch --delete-conflicting-outputs

Очистить и перегенерировать

dart run build_runner clean dart run build_runner build --delete-conflicting-outputs

Генерация только для конкретного файла

dart run build_runner build --build-filter="lib/models/**"

▶️ Запуск

Список устройств

flutter devices

Запуск на устройстве по умолчанию

flutter run

Запуск на конкретном устройстве

flutter run -d <device_id>

Удалить приложение с устройства

flutter install --uninstall

Hot reload (внутри flutter run)

r # hot reload R # hot restart q # quit

📱 Сборка APK

Debug APK

flutter build apk --debug

Release APK

flutter build apk --release

Split APK по архитектуре (меньше размер)

flutter build apk --split-per-abi

App Bundle (для Google Play)

flutter build appbundle --release

🛠️ Решение проблем

Пересоздать Android-часть (если gradlew сломан)

flutter create --platforms=android .

Проверить окружение

flutter doctor -v

Проверить Java версию

java -version
💡 Должна быть Java 17 (идёт с Android Studio в папке jbr)

Принудительная очистка Gradle wrapper

cd android ./gradlew clean cd .. flutter clean

⚡ ВСЁ В ОДНОМ (копируй целиком)

Полная очистка + переустановка + запуск

flutter clean && \ rm -f pubspec.lock && \ rm -rf android/.gradle android/build android/app/build android/local.properties && \ rm -rf .dart_tool .flutter-plugins .flutter-plugins-dependencies .idea && \ taskkill //F //IM java.exe 2>/dev/null ; \ taskkill //F //IM gradle.exe 2>/dev/null ; \ rm -rf ~/.gradle/caches ~/.gradle/daemon ~/.gradle/wrapper/dists && \ dart pub cache repair && \ flutter pub get && \ dart run build_runner build --delete-conflicting-outputs && \ flutter run
⚠️ Первый запуск после такой очистки: 10-20 минут

📥 Автоподбор версий (без конфликтов)

💡 Если pub get ругается на версии — используй эти команды. Pub сам подберёт совместимые.
